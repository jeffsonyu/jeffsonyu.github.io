Selected Projects

Visual-Tactile in-hand Object reconstruction (with Hand), VTacO & VTacOH

Paper
Webpage
Code
Tactile sensing is one of the modalities humans rely on heavily to perceive the world. Working with vision, this modality refines local geometry structure, measures deformation at the contact area, and indicates the hand-object contact state. In this project, we aim to leverage tactile information with visual feature, to reconstruct both rigid and deformable objects.
  • We construct a visual-tactile learning framework to reconstruct an object when it is being manipulated. We provide the object-only version VTacO, and the hand-object version VTacOH.
  • we propose a simulation environment, VT-Sim, which supports generating hand-object interaction for both rigid and deformable objects.
  • Our method outperforms the previous baselines quantitatively and qualitatively, and we also validate our method in the real world data.

Tactile Needle Threading, T-NT

Paper
Webpage
Code
We propose a Tactile perception-based approach to address the Needle Threading task and name it T-NT. This task is divided into two main stages: Tail-end Finding and Tail-end Insertion. In the first stage, the agent traces the contour of the thread twice using vision-based tactile sensors mounted on the gripper fingers, to locate the tail-end of the thread. In the second stage, it employs a tactile-guided RL model to drive the robot to insert the thread into the target needle eyelet. Extensive experiments on real robots are conducted to demonstrate the efficacy of our method.

ViTaM-D

Paper (Coming Soon)
Webpage (Coming Soon)
Code (Coming Soon)
We present ViTaM-D, a novel visual-tactile framework for dynamic hand-object interaction reconstruction, integrating distributed tactile sensing for more accurate contact modeling.
  • We leverages distributed tactile sensors to address this limitation by introducing DF-Field. This distributed force-aware contact representation models both kinetic and potential energy in hand-object interaction.
  • ViTaM-D first reconstructs hand-object interactions using a visual-only network, VDT-Net, and then refines contact details through a force-aware optimization (FO) process, enhancing object deformation modeling.
  • To benchmark our approach, we introduce the HOT dataset, which features 600 sequences of hand-object interactions, including deformable objects, built in a high-precision simulation environment.
